Modificación de módulos

Desde estas opciones se puede realizar cualquier tipo de modificación en los módulos de esta modalidad.

Además, el coordinador de una modalidad académica, de acuerdo con la jerarquía de roles, puede desmpeñar las tareas de un profesor principal. Por lo que puede administrar los módulos de su modaliada a través de la opción que se brinda por cada módulo.

  • Módulos (Ver/Eliminar):

    Mediante esta opción podemos conocer el listado de módulos de esa modalidad académica, mostrando datos como: su código, el nombre, el estado, profesor principal y el correo electrónico del profesor Así como eliminar cualquiera de ellos primero marcándolo y luego presionando la opción Eliminar.


  • Adicionar módulo:

    Cuando se necesite adicionar un módulo a esta modalidad se realiza a través de esta opción donde se especifica el nombre del nuevo módulo, el estado se selecciona uno dentro de la opciones que se brindan a continuación.
    • No visible: el módulo no estará visible para el alumno.
    • Deshabilitado: el alumno solo tendrá acceso a las generalidades del módulo, pero las lecciones se encontrarán deshabilitadas.
    • En curso: el alumno tiene acceso tanto a las generalidades del módulo como a las lecciones que se le ofrecen.
    • Finalizado: se tiene acceso a las generalidades del módulo y a sus datos históricos.

    Además, se seleciona el profesor principal de ese módulo presionando el botón . Por último, se procede a adicionar el nuevo módulo con todos sus datos en la base de datos del sistema.
    Una vez creado el módulo el sistema le asigna un código con el cual el profesor principal desde la herramienta de publicación puede publicar las lecciones, materiales y autoevaluaciones especificando dicho código. Para más información consulte el manual de la herramienta de publicación SEPADHP versión 1.5
