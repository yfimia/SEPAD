Modificación del alumnado

Desde estas opciones se puede realizar cualquier tipo de modificación en los alumnos y grupos de alumnos de esta modalidad académica.

  • Alumnado (Ver/Eliminar):

    Mediante esta opción podemos conocer el listado de alumnos de esa modalidad académica. Así como eliminar cualquiera de ellos primero marcándolo y luego presionando la opción Eliminar.


  • Adicionar alumno:

    Cuando se necesite adicionar un alumno a esta modalidad se realiza a través de esta opción donde se listan todos los usuarios que están matriculados en el sistema. Se marca el alumno a agregar y se presiona la opción Adicionar.



  • Adicionar grupo de alumnos:

    Mediante esta opcíón se listan los grupos de alumnos existentes en el sistema y sus descripciones para adicionarse a la modalidad. Primero se marca el grupo a agregar y luego se presiona Adicionar.



  • Solicitudes de matrícula:

    Para aceptar o denegar las solicitudes de matrícula para esta modalidad de debe realizar a través de esta opción. Primero se ha de marcar el nombre que se desea y si se quiere aceptar la solicitud presione la opción Matrícula. Por otra parte, para denegar la solicitud presione Denegar.
