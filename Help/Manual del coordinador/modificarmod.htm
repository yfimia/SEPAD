Modificar generalidades de modalidades

Desde esta opción se puede realizar cualquier tipo de variación en la modalidad académica o simplemente ver el estado de sus campos, como se muestra en la figura siguiente:

Los campos a modificar son:

  • Nombre: Este campo permite editar el nombre de la modalidad que se esté tratando.
  • Estado: Se selecciona el estado que se necesite dentro de las opciones que se brindan:
    • No visible: la modalidad académica no estará visible para el alumno.
    • Deshabilitado: no se tiene acceso a los módulos de esta modalidad, ni se permite solicitar matrícula a ella.
    • En inscripción: se puede matricular en esta modalidad mediante la opción Solicitar matrícula. No se pueden acceder los módulos.
    • En curso: No se permite solicitar matrícula, pero se tiene acceso a todos los módulos de esta modalidad.
    • En inscripción-en curso: es una unión de los estados En inscripción y En curso. En él se permite matricular a través de la opción Solicitar matrícula y además se tiene acceso a los módulos de esa modalidad académica.
    • Finalizado: Se tiene acceso a los datos históricos de la modalidad y no se puede acceder a los modulos
  • Coordinador: No permite editar. Para escoger un nuevo coordinador de modalidad se debe presionar el botón
  • Claustro: Para escoger un nuevo claustro se persionar el botón . Si desea crear un nuevo grupo de profesores se debe escribir el nombre del nuevo grupo y marcar la siguiente opción
  • Grupo de alumnos: En el botón se escogen los alumnos que van a recibir esta modalidad. Si se desea crear un nuevo grupo de alumnos se escribe el nombre del grupo nuevo y se debe marcar la opción

Nota: los demás campos si tienen permiso para editar.

  • Certificados.
  • Objectivos generales.
  • Programa.
  • Fecha y duración.
  • Perfil de egresado.
  • Sistema de tutorías.
  • Recursos.
  • Sistema de evaluación.
  • Coste.
  • Datos de la institución.
  • Fundamentación.
  • Sede.
  • Bibliografía general.

Si se modificó alguno de los campos para que ellos se actualicen en la base de datos se debe dar click en el botón
