El usuario administrador

El usuario con privilegios de administrador, como se ve en la figura en sus datos generales tiene Permisos : Administrador. Este podrá acceder a todas las partes del sistema.


Para acceder a la consola de administración en la barra de servicios  la opción Administración. Desde allí se podrán configurar las distintas partes del sistema, así como manipularlas.

El administrador debido a la jerarquía de roles puede desempeñar las funciones de un coordinador de modalidad, de un profesor principal o de un tutor, por ende tiene acceso a todas las consolas de cada uno de estos roles.
