Manipulador de usuarios
  • Crear usuario:

    Para adicionar un usuario se debe escribir sus datos como siguen a continuación:
    • Nombre y apellidos
    • Identificador
    • Contraseña
    • Confirmar contraseña
    • Tipo de usuario
    • Email
    • Fecha de nacimiento
    • Sexo
    • Dirección
    • Código postal
    • Teléfono
    • País
    • Provincia
    • Título
    • Institución
    A través de esta la opción Enviar correos a la cuenta de correo externa se podría enviar mensajes a la cuenta de correos externa al sistema especificada en el perfil del usuario.
    Es obligatorio llenar todos los campos que están en negrita. No se aceptaran usuarios con el mismo identificador.



  • Usuario:

    Se muestra una lista con el identificador, el nombre y el correo electrónico de todos los usuarios matriculados en el sistema y se utiliza para cualquier modificación que se quiera o se necesite hacer de un usuario ya creado.
    Para crear un usuario se presiona la opción Crear, donde se llenan los datos de dicho usuario.
    Para eliminar un usuario se marca el usuario que se desee y se presiona la opción Eliminar.



  • Crear grupo:

    Esta opción le posiblita al administrador adicionar nuevos grupos. Se especifica el nombre y la descripción del grupo a crear y se presiona la opción Adicionar.


  • Grupos de usuarios:

    Muestra una lista con el nombre y la descripción de los grupos creados en el sistema. De cada grupo tenemos acceso a la lista de los usuarios que lo conforman.
    Para crear un grupo nuevo se presiona la opción Crear, donde se llenan los datos de dicho grupo.
    Para eliminar se marca el grupo que se desee y se presiona la opción Eliminar.


  • Adicionar usuarios a un grupo:

    Se selecciona el grupo al cual se le agregará el usuario, posteriormente se marca el usuario dentro de la lista de usuarios que se muestran y se presiona la opción Adicionar. Nótese que un mismo usuario puede pertenecer a varios grupos.


  • Eliminar usuarios de un grupo:

    Dada la lista de usuarios con el nombre del grupo a que pertenecen, se marca el usuario que se desee y se presiona la opción Eliminar.
