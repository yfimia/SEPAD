Configuración general del sistema


  • Configuración del servidor de base de datos

    Desde esta opción se podrá configurar el servidor de base de datos especificando:
    • DSN: nombre de la fuente de datos.
    • Tipo de servidor: tipo del servidor de base de datos a utilizar, se proponen dos opciones Microsoft Access y SQL Server.



  • Configuración del servicio de notificación automática a la solicitud de registro en el sistema.

    Desde esta opción se podrá configurar una cuenta de correo para notificar automáticamente cuando se acepta un usuario en el sistema. Los datos a configurar son los siguientes:
    • Dirección de correo: Dirección con la cual se enviará el mensaje de notificación.
    • Usuario: Usuario para conectarse a la cuenta a través del protocolo de correo electrónico SMTP.
    • Contraseña: Contraseña para conectarse a la cuenta a través del protocolo de correo electrónico SMTP.
    • Director del servidor de correo: Dirección del servidor de SMTP por ej. uclv.edu.cu o 172.20.17.3.
    • Puerto: Puerto para conectarse al servidor de correo. Casi siempre es 25.
    • Autentificación: Si se marca Si el servidor requiere autentificación, si se marca No no la requiere.
    • Asunto: Asunto que tendrá el mensaje de notificación.
    • Cuerpo del mensaje de respuesta a solicitudes de registro al sistema: Cuerpo  que tendrá el mensaje de notificación.



  • Configuración general.

    Además se pueden configurar otras opciones de carácter general como:
    • Logotipo: Si se desea cambiar el logotipo que se muestra en la esquina superior izquierda, se puede cambiar desde esta opción. El camino debe ser o relativo a una carpeta dentro del directorio del sistema por ej. ../images/logo.jpg o un camino absoluto como http://www.uclv.edu.cu/images/logo.jpg.
    • Página principal : Si se desea cambiar la página principal que se muestra cuando se entra al sistema, se puede cambiar desde esta opción. El camino debe ser relativo a una carpeta dentro del directorio del sistema por ej. ../beta/homepag.htm o un camino absoluto como http://www.uclv.edu.cu/pages/homepage.htm.
    • Cantidad de elementos de las listas que se muestran en el sistema: Cuando los elementos que se muestran en el sistema en forma de listas son muchos, es conveniente mostrarlos por lotes para optimizar la velocidad de respuesta del sistema, el tamaño de estos lotes es lo que se define en esta opción.
    • Cantidad de día para considerar una noticia como nueva: Las noticias que tengan menos días que los especificados aquí, se considerarán nuevas y aparecerán marcadas como tal con una imagen
