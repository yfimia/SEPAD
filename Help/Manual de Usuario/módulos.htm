Módulos

Los módulos están constituidos por un conjunto de lecciones a las cuales se les asocian materiales y auto evaluaciones. Los módulos están asociados a una modalidad y al igual que ellas tienen varios estados.



Las generalidades del módulo cuenta con los siguientes datos:

  • Nombre del módulo
  • Código
  • Estado
    • No visible: el módulo no estará visible para el alumno.
    • Deshabilitado: el alumno solo tendrá acceso a las generalidades del módulo, pero las lecciones se encontrarán deshabilitadas.
    • En curso: el alumno tiene acceso tanto a las generalidades del módulo como a las lecciones que se le ofrecen.
    • Finalizado: se tiene acceso a las generalidades del módulo y a sus datos históricos.
  • Claustro: Se muestra un listado del identificador, nombre y apellido y correo electrónico del profesorado que atiende el módulo. Dentro del claustro se asigan un profesor para que desempeñe la función de profesor principal, este es el encargado de la organización y la dirección de ese módulo en particular, el resto de los profesores que alli se muestran son los tutores.
  • Palabras claves: se muestran las palabras claves del módulo, y estas pueden ser modificadas, eliminadas o se puede además adicionar otras.
  • Calendario de actividades: la lista de actividades con fecha y hora de realización. se puede modificar o eliminar cualquiera de las actividades planificadas o adicionar otras.
  • Objetivos generales: son los objetivos que persige el módulo. Se puede eliminar, modificar cualquier objetivo o adicionar algún otro.
  • Sistema de contenidos.
  • Conocimientos requeridos: se muestran los conocimientos que deben de tenerse para cursar este módulo. Se puede modificar, eliminar o adicionar otros elementos.
  • Requerimientos técnicos y de software: se listan los requerimientos técnicos y de software que se necesiten. También, permite modificar o eliminar cualquiera de estos requerimientos o adicionar algún otro.
  • Orientaciones generales.
  • Fundamentación.
  • Programa.
  • Recursos.
  • Sistema de tutorías.
  • Bibliografía general.
