Calificaciones

A través del formulario de consultas se pueden obtener las calificaciones de un rango de fechas determinado. Tambien se le permite al usuario que pueda escoger el tipo de calificaciones que quiere listar. Por defecto las muestra todas.

Al presionar el botón Consultar se le muestra al estudiante un listado con el nombre de la lección en que fue evaluado, la fecha y la hora así como la calificación obtenida. Cada estudiante solo tendrá acceso a conocer sus calificaciones.
